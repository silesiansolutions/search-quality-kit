# v0.7 delivered scope

v0.7 operates the existing checks across more sites without expanding the SEO rule catalog:

- monorepo and multi-site runner;
- aggregated reports with per-site attribution;
- per-site baselines and isolated failure policy;
- a public in-repository HTTP showcase and workflow;
- per-site SARIF without fabricated aggregate locations;
- optional Search Console integration as a separate package/plugin, never a core dependency.

The portfolio runner, reports, baselines, Action mode, and public showcase shipped in v0.7. A standalone docs site, trend storage, and richer annotations remain future options. The package does not call Google APIs, execute browsers, score content, or measure Core Web Vitals.
